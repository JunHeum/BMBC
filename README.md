# BMBC

Junheum Park,
Keunsoo Ko, 
Chul Lee,
and Chang-Su Kim

Official PyTorch Code for **"BMBC: Bilateral Motion Estimation with Bilateral Cost Volume for Video Interpolation"** 

### Requirements
- PyTorch 1.3.1 (We tested 1.4.0 and got results with errors)
- CUDA 10.0
- CuDNN 7.6.5
- python 3.6
